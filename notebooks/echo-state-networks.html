Echo State Networks (ESNs)

  • Big Picture

    • Herbert Jaeger. "Echo state network." Scholarpedia 2.9 (2007): 2330. link

    • Herbert Jaeger. "The ''echo state'' approach to analysing and training recurrent neural networks-with an erratum note." Bonn, Germany: German National Research Center for Information Technology GMD Technical Report 148.34 (2001): 13. link

      • Gives rationale for the 'echo' in echo state network, and provides several equivalent definitions of the echo state property. Provides an illuminating example of learning a periodic-but-not-sinusoidal signal by entraining the reservoir nodes to a sinusoid, giving a diverse set of periodic-but-not-sinusoidal signals in the reservoir nodes.

    • M. Lukoševičius, "A Practical Guide to Applying Echo State Networks," in Neural Networks: Tricks of the Trade, 2nd ed., vol. 7700, no. 2, Berlin, Heidelberg: Springer, Berlin, Heidelberg, 2012, pp. 659–686. link

    • G. Manjunath and H. Jaeger, "Echo state property linked to an input: Exploring a fundamental characteristic of recurrent neural networks," Neural Computation, vol. 25, no. 3, pp. 671–696, Mar. 2013.

      • A more formal presentation of the echo state property in the context of non-autonomous dynamical systems. The definition of the echo state property is especially clear here. For example, the authors explicitly point out that, if a system has the echo state property, then for any given bi-infinite input sequence, there is one and only one bi-infinite state sequence that is admitted by that input sequence. This fact can be deduced from the original definition of the echo state property, but this paper makes it an explicit proposition with a proof.
      • This paper also gives the cleanest requirements on the input and state space, namely the input space should be a complete metric space, and the state space should be a compact metric space. Though, a point of confusion: under the hyperbolic tangent, \(\tanh : \mathbb{R} \to (-1, 1) \), which isn't a compact metric space...
      • Also does a good job of refuting the 'folk lore' that has sprung up around the echo state property, i.e. the claim that taking the spectral radius of the reservoir matrix less than 1 ensures the echo state property. This is a folk lore that I myself fell prey to in an early paper.

  • Closeup

    • C. Gallicchio, "Chasing the Echo State Property." 27-Nov-2018. arXiv preprint
      • Undertakes an empirical investigation of the echo state property as a function of various common rescalings of the reservoir matrix (\mathbf{W}) and input matrix (\mathbf{W}_{\text{in}}). Finds that the appropriate rescalings are typically much larger than common practice in the literature, and thus "[suggests] that the actual domain of ESP validity is much wider than what covered by literature conditions commonly used in RC practice." The author uses one-step-ahead prediction on the Santa Fe laser time series and the sunspot time series.
