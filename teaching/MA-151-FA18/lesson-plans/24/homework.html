Section 9.3:

For each of the following problems:

  1. Use Minitab to generate a scatter plot with a line of best fit:
    • From the top menu bar, select Stat > Regression > Fitted Line Plot....
    • Select the appropriate response and predictor.
    • Select Linear for the Type of Regression Model.
  2. Print out the scatter plot.
  3. Annotate the print out with a description of the trend and its strength.
  4. Use Minitab to estimate the linear regression model using Stat > Regression > Fit Regression Model....
  5. Print the session window.
  6. Annotate the printout with the estimate of the linear regression model.
  7. Annotate the printout with your interpretation of the slope and intercept of the linear regression model.
  8. Annotate the printout with your interpretation of the Root-Mean-Square Error (RMSE) / Standard Error of Prediction. Minitab calls this S.
  9. Annotate the printout with the best prediction for the response variable at the specified value of the predictor variable.

NOTE: For problems 9.3.16 and 9.3.17, switch the direction of prediction: predict the cholesterol level (CHOL) of a woman using either Body Mass Index (BMI) or weight (WT). So the response is the cholesterol level, and the continuous predictor is either the BMI or the weight.
