Section 9.1: Tests of a Hypothesis Based on a Single Sample

  1. Define a statistical hypothesis test.
  2. State what types of quantities are of interest in statistical hypothesis tests.
  3. Given a claim stated in words, determine a relevant population parameter related to the claim, and write the claim as an equality / inequality in terms of that population parameter.
  4. Explain what the null and alternative hypotheses correspond to in a hypothesis test.
  5. Given a claim stated in words, determine whether the claim is a null hypothesis or an alternative hypothesis, and determine the complementary claim.
  6. Explain the main approach of a hypothesis test as finding evidence either for or against each of the null hypothesis and the alternative hypothesis.
  7. Define the Type I Error Rate of a hypothesis test.
  8. Define the significance level (‘level’) of a hypothesis test.
  9. Relate the Type I Error Rate of a hypothesis test to the confidence level of an interval estimator used to test the hypothesis.

Section 9.2: Tests About a Population Mean

  1. State and perform the procedure for testing a pair of hypotheses of the form \(H_{0} : \mu = \mu_{0}\) vs. \(H_{a} : \mu \neq \mu_{0}\) using a confidence interval.
  2. State and perform the procedure for testing a pair of hypotheses of the form \(H_{0} : \mu \leq \mu_{0}\) vs. \(H_{a} : \mu > \mu_{0}\) using a confidence bound.
  3. State and perform the procedure for testing a pair of hypotheses of the form \(H_{0} : \mu \geq \mu_{0}\) vs. \(H_{a} : \mu < \mu_{0}\) using a confidence bound.

Section 9.3: Tests Concerning a Population Proportion

  1. State and perform the procedure for testing a pair of hypotheses of the form \(H_{0} : p = p_{0}\) vs. \(H_{a} : p \neq p_{0}\) using a confidence interval.
  2. State and perform the procedure for testing a pair of hypotheses of the form \(H_{0} : p \leq p_{0}\) vs. \(H_{a} : p > p_{0}\) using a confidence bound.
  3. State and perform the procedure for testing a pair of hypotheses of the form \(H_{0} : p \geq p_{0}\) vs. \(H_{a} : p < p_{0}\) using a confidence bound.
